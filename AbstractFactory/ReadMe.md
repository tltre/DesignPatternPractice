## 命令行启动 FJE
### 树形结构 + 图标族 poker-face
```shell
fje -f ../../test.json -s tree -i poker-face
```
### 矩形结构  + 图标族 poker-face
```shell
fje -f ../../test.json -s rect -i poker-face
```
### 树形结构 + 图标族 tri-circle
```shell
fje -f ../../test.json -s tree -i tri-circle
```

### 矩形结构 + 图标族 tri-circle
```shell
fje -f ../../test.json -s rect -i tri-circle
```
